# ⏱️ Flutter Time Tracker

A comprehensive, professional time-tracking application built with Flutter. This app helps individuals and teams monitor their productivity through real-time tracking, detailed statistics, and organized project management.

---

## 🚀 Features

- **Real-time Time Tracking**: Start, pause, and stop timers for specific projects and tasks.
- **Project & Task Management**: Easily create and manage projects and associated tasks.
- **Data Visualization**: Interactive charts (Pie & Bar) to visualize time distribution and daily activity.
- **Advanced Sorting & Filtering**: Organize your entries by date, duration, or project name. Filter entries to focus on specific projects.
- **Data Export**: Export your time tracking logs to **CSV** format for easy sharing and external analysis.
- **Clean UI/UX**: Modern, card-based interface with smooth animations and intuitive navigation.
- **State Management**: Built with `Provider` for efficient and scalable state handling.

## 🛠️ Tech Stack

- **Framework**: [Flutter](https://flutter.dev/)
- **State Management**: [Provider](https://pub.dev/packages/provider)
- **Charts**: [fl_chart](https://pub.dev/packages/fl_chart)
- **ID Generation**: [uuid](https://pub.dev/packages/uuid)
- **Data Export**: [csv](https://pub.dev/packages/csv) & [share_plus](https://pub.dev/packages/share_plus)
- **Storage**: [localstorage](https://pub.dev/packages/localstorage)

## 📦 Getting Started

### Prerequisites

- Flutter SDK (latest stable version)
- Dart SDK
- An IDE (VS Code, Android Studio, or IntelliJ)

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/SatyamIT86/time_tracker.git
   ```

2. **Navigate to the project directory:**

   ```bash
   cd time_tracker
   ```

3. **Install dependencies:**

   ```bash
   flutter pub get
   ```

4. **Run the application:**
   ```bash
   flutter run
   ```
